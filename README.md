# 🚀 Clack Portfolio

A world-class, premium portfolio website showcasing expertise in **Software Engineering** and **Cybersecurity**. Built with a focus on high-performance animations, secure architecture, and modern aesthetics.

---

## ✨ Features

- **Dynamic Hero Section**: High-impact introduction with fluid animations.
- **Cybersecurity Showroom**: Dedicated sections for Offensive and Defensive security skills.
- **Project Gallery**: interactive display of production-grade systems.
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop.
- **Modern Tech Stack**: Leveraging React, Vite, Tailwind CSS, and Framer Motion.

## 🚀 Getting Started

To run this project locally:

1.  **Clone the repository**:
    ```bash
    git clone https://github.com/odongoclack/clack-portfolio.git
    ```
2.  **Install dependencies**:
    ```bash
    npm install
    ```
3.  **Start the development server**:
    ```bash
    npm run dev
    ```
